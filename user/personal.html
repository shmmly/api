# 个人中心

# 个人信息接口

功能： 返回用户的一些个人信息：用户名、头像等

  • 提供参数：用户id
  • 返回参数：用户的信息

TIP

已完成，头像信息暂无

# 用户出行总数和里程数接口

功能： 返回用户所有的出行总次数、以及总的里程数

  • 提供参数：用户id
  • 返回参数：出行次数、总里程数

TIP

已完成，总里程数暂无

# 用户评价的个数以及未评价个数

功能： 获取用户已经评价和未评价的个数

  • 提供参数：用户id
  • 返回参数：评价个数、未评价次数

TIP

已完成

# 修改用户昵称接口

功能： 可以修改用户的昵称

  • 提供参数：用户id 修改之后的名称
  • 返回参数：是否成功

TIP

已完成

# 用户密码修改接口

已完成
