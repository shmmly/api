# 我的申请

# 正在申请中的车辆接口

功能： 用户在用车申请页面，能看到所有的用车申请记录（申请中）.这是一个需要分页的接口。

  • 提供参数：用户id、分页个数、申请状态
  • 返回参数：按照时间排序的分页数组

TIP

已完成 等同于未评价接口

# 已经完成的申请车辆接口

功能： 用户在用车申请页面，能看到所有的用车申请记录（已完成）.这是一个需要分页的接口。

  • 提供参数：用户id、分页个数、申请状态
  • 返回参数：按照时间排序的分页数组

TIP

已完成 等同于未评价接口

# 单个车辆申请的详情接口

功能： 用户点击简略的申请信息之后，会跳转到这个记录的申请详情页面。

  • 提供参数：用户id、申请记录id
  • 返回参数：完整的申请详情。

TIP

已完成

# 已经完成申请记录的删除接口

功能： 这里采用逻辑删除

  • 提供参数：用户id、申请记录id
  • 返回参数：是否删除。

TIP

已提供未提交申请的删除功能 完成申请的暂不提供删除功能
