# 请假

::: waring 司机请假 暂不支持工作流 目前功能增删改查 :::

# 申请请假接口

功能： 司机填写内容，申请请假

-提供参数：开始时间，结束时间，请假类型，图片，司机id -返回结果：true/false

TIP

已完成

# 请假记录列表接口

功能： 展示司机所有请假记录

-提供参数：司机id -返回结果：请假类型，开始时间，结束时间，申请时间，申请状态

TIP

已完成

# 请假详情接口

功能： 展示请假详情

-提供参数：司机id，请假id -返回结果：请假类型，开始时间，结束时间，时长，请假事由，司机姓名，申请时间，申请状态

TIP

已完成

# 修改请假内容接口

功能： 修改请假内容

-提供参数：开始时间，结束时间，请假类型，图片，司机id，请假id -返回结果：true/false

TIP

已完成
