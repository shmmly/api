# 当前任务接口

功能： 司机目前正在完成和未完成任务列表，根据时间进行排序

-提供参数：司机id，任务状态 -返回结果：起始时间，结束时间，出发地，目的地，状态，用车人联系方式

TIP

已完成

# 历史任务接口

功能： 司机已经完成的任务列表

-提供参数：司机id，任务状态 -返回结果：起始时间，结束时间，出发地，目的地，状态，用车人联系方式

TIP

已完成

# 失效任务接口

功能： 申请取消任务列表

-提供参数：司机id，任务状态 -返回结果：起始时间，结束时间，出发地，目的地，状态，用车人联系方式

# 任务详情接口

功能： 展示当前任务详情

-提供参数：司机id，任务id -返回结果：任务状态，出发地，目的地，起始时间，结束时间，申请人姓名，联系方式，用车人数，出行事由，分派车牌（在派车中的状态下）
