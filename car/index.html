# 搜索接口

功能： 司机可以通过起始地址|目的地|出发时间|结束时间|车牌号 等条件来筛选获取想要的任务记录。

  • 提供参数：起始地址|目的地|出发时间|结束时间|车牌号 其中某一个
  • 返回结果：按照时间排序的数组

danger

废弃

# 轮播图接口

功能： PC端可以通过后台来维护这个接口需要展示的内容，用来通知用户某些信息等.比如：五一休假不提供车辆等等。

  • 提供参数：无
  • 返回参数：图片base64码以及必要的文字内容

danger

静态写死，后续版本讨论

# 展示待办任务列表接口

功能： 展示司机待办任务列表，按时间进行排序，只取前面几个。

-提供参数：司机id，任务状态，数据个数 -返回参数：任务id，开始时间，结束时间，出发地，目的地，状态，用车人联系方式

TIP

已完成

# 司机申请取消任务接口

功能： 司机可以通过点击取消任务按钮申请取消任务安排

-提供参数：司机id，当前任务id

danger

暂不提供该功能，默认接受任务安排
